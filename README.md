# Reviews-Report
Система автоматизированного сбора, обработки и анализа отзывов пациентов сети клиник. Проект представляет собой решение на базе Google Apps Script, интегрированное с Google Sheets, предназначенное для формирования единой базы отзывов из внешних источников и генерации аналитических отчетов.
